Kakaomap shortcut
=================

THIS IS NOT AN OFFICIAL PLUGIN OF MAP.KAKAO.COM

이 플러그인은 카카오에서 만드는 공식 플러그인이 아닙니다.

* 이 플러그인은 카카오 맵을 좀 더 쉽게 사용할 수 있는 플러그인을 추가합니다.
* 개인 정보를 수집하지 않으며, 사용자 행동을 수집하지 않습니다.


SHORTCUTS
=========
* 지도설정
Shift+Ctrl+1 : 교통정보
Shift+Ctrl+2 : CCTV
Shift+Ctrl+Q : 사고.공사
Shift+Ctrl+W : 자전거
Shift+Ctrl+A : 지형도
Shift+Ctrl+S : 지적편집도
Shift+Ctrl+Z : 법정경계
Shift+Ctrl+X : 행정경계

* 지도타입
Shift+Ctrl+M : 지도 / 스카이뷰 토글

* 확대/축소
Alt+"+" : 확대 (Window)
Alt+"-" : 축소 (Window)
Cmd+"+" : 확대 (맥)
Cmd+"-" : 축소 (맥)
